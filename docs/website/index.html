🎬 Démo

Comment ça marche ?

  1. Uploadez votre fichier APK
  2. Choisissez le mode de transformation
  3. Configurez le nom et Package ID
  4. Transformez et téléchargez !

💡 Astuce : Utilisez des APK non protégés pour de meilleurs résultats.
