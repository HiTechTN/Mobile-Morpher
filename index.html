⚡ Installation Rapide

docker compose up -d

📱 Utilisation

1

Uploader un APK

Glissez-déposez ou sélectionnez un fichier APK sur l'interface web

2

Choisir le mode

  • Express: Transformation rapide (nom + package)
  • Design: Personnalisation complète
  • Developer: Accès au code Smali
3

Configurer

Entrez le nouveau nom de l'application et le Package ID

4

Transformer

Cliquez sur "Transformer" et attendez la compilation
